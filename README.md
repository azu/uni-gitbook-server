# uni-gitbook-server

[Unofficial] Alternative of GitBook server.

## Install

Install with [npm](https://www.npmjs.com/):

    npm install uni-gitbook-server -g

## Usage

```
$ uni-gitbook-server --bookSrouce /path/to/project-dir/
```

Install devDependencies and Run `npm test`:

    npm i -d && npm test
